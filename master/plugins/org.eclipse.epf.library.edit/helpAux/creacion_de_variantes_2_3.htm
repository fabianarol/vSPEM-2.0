2.3 Creación de variantes

Top  Previous  Next

Para la creación de variantes se siguen los mismos pasos que se han seguido dentro de la creación de puntos de variación. Situándose dentro un proceso haremos click derecho sobre el proceso y se elige por ejemplo Var Iteration:

Y se desplegará una ventana de un diseño similar a la de los puntos de variación:

La ventana de creación de variantes tiene los siguientes campos:

•Name:

Nombre que se le dará a la variante.

•Description:

Una breve descripción de porque se genera dicha variante.

•Icon:

El icono por defecto representativo para esa variante.

•Dependencies with var points(Inclusive):

Dependencias de inclusividad con otros puntos de variación, si añadimos puntos de variación aquí significará que añadir esta variante provocará que se tengan que añadir los puntos de variación de la lista en la adaptación de un proceso. Esta variante incluye el uso de los puntos de variación seleccionados en la adaptación de un proceso.

•Dependencies with variants(Inclusive):

Dependencias de inclusividad con otras variantes, si añadimos variantes a esta lista significará que si añadimos esta variante habrá que añadir además el resto de variantes al punto de variación ocupado en la adaptación de un proceso. Esta variante incliye el uso de las variantes seleccionadas en la ocupación de un punto de variación.

•Dependencies with variants(Exclusives):

Dependencias de exclusividad con otras variantes, si añadimos puntos de variación en esta lista querrá decir que provocaremos conflictos si añadimos el punto de variación creado y además alguno de los de la lista añadidos. Esta variante excluye el uso de las variantes seleccionadas en la ocupación de un punto de variación.
