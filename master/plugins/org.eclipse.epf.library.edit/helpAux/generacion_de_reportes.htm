2.6.1 Generación de reportes

 Top  Previous 

vEPFC tiene una funcionalidad extra para poder generar reportes en .pdf actualmente. El reporte que se puede generar actualmente es del proceso adaptado y muestra un resumen ilustrativo de los elementos que se han utilizado para su adaptación.

Para generar un reporte se pulsará con el botón derecho sobre el proceso adaptado (una vez finalizada la adaptación).

Al pulsar sobre la opción de generar reporte (Generate Report) se abre una ventana donde debemos introducir el nombre del archivo que queremos dar al reporte y el formato .pdf.

Una vez introducido el nombre la interfaz validará si es un nombre valido. Cargará el reporte y obtendremos una ventana de confirmación al crearlo.
