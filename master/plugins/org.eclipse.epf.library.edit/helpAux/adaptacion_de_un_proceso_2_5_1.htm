2.5.1 Adaptación de un proceso

Top  Previous  Next

Para realizar la adaptación de un proceso se debe hacer click derecho sobre el proceso adaptado creado y pulsar sobre la opción de adaptar (Adapt Process).

Se desplegará una interfaz donde el usuario trabajará para crear las diferentes variaciones y solución de conflictos y colisiones. La ventana tiene la siguiente forma.

En ella se distinguen los siguientes campos:

•Name:

Nombre de la Línea de Proceso.

•Description:

Descripción a añadir a la adaptación de proceso.

•Help (Botón):

Botón de ayuda que desplegará un manual de usuario.

•Variations completed:

Lista en forma jerarquica de las variaciones que se llevan actualmente completadas.

•Variations Points Pending:

Puntos de variación de los que se disponen para generar variaciones.

•Cardinality:

Lista que muestra las colisiones por cardinalidad.

•Conflicts:

Lista que muestra los conflictos por dependencias.

Para generar variaciones se selecciona un punto de variación y se pulsa sobre el botón de añadir variación (Add Variation). Se abre una ventana de la siguiente forma.

Esta ventana tiene los siguientes campos:

•Documentation:

Una breve descripción de la razón de la variación.

•Name:

Nombre del punto de variación que se eligió.

•Description:

Descripción del punto de variación.

•Min cardinality:

Cardinalidad mínima del punto de variación.

•Max cardinality:

Cardinalidad máxima del punto de variación.

•Type:

Tipo de relación de cardinalidad.

•Variant:

Lista de variantes disponibles para añadir al punto de variación, dependerá de si el punto de variación es abierto y de su compatibilidad.

•Conflict:

Conflictos que se producen al añadir ese punto de variación. Los conflictos pueden ser con otras variantes, o de exclusividad con el propio punto de variación.

Existen una serie de botones que ayudan al usuario a gestionar las dependencias y tener una misión global de la variación que se está construyendo. Si no existe ninguna dependencia se puede generar la variación. Para ver un ejemplo se generan 3 variaciones.

Si se desea se puede guardar el estado del proceso y continuar mas tarde o por el contrario si no se tiene ningún conflicto/colisión se puede generar la adaptación. Para generar la adaptación del proceso se debe pulsar el botón Ok.
