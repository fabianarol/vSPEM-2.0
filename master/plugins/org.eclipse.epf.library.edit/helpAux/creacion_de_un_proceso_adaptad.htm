2.5 Creación de un proceso adaptado

Top  Previous  Next

Para crear un proceso adaptado se debe tener al menos una Variante para poder generar un proceso adaptado (Tailored Process Component). Para crear el proceso adaptado el usuario se debe situar sobre la carpeta de procesos adaptados (Tailored Processes).

Haciendo click derecho sobre dicha carpeta se selecciona la opción de un nuevo proceso adaptado.

Se abrirá una ventana donde se pedirá introducir ciertos datos, como el nombre y la configuración del proceso adaptado (Esta configuración por defecto es la de la línea de procesos).

Pulsado el botón Ok se genera el proceso adaptado con el nombre justo debajo de la carpeta de Procesos Adaptados y se abrirá la ventana de edición para introducir datos extra si así se desea. Se puede ver la arquitectura en la siguiente imagen.

Dicha estructura tiene las siguientes carpetas:

•Capability Patterns:

En esta carpeta se copiarán los patrones de capacidad del Core Process una vez finalizada la adaptación del proceso, en su interior se formarán las variaciones creadas junto con los elementos SPEM creados independientemente.

•Delivery Processes:

En esta carpeta se copiarán los procesos de entrega del Core Process una vez finalizada la adaptación del proceso, en su interior se formarán las variaciones creadas junto con los elementos SPEM creados independientemente.

•Variations:

En esta carpeta se situarán las variaciones generadas en la adaptación del proceso a modo de jerarquía, dando al usuario una visión global de las variaciones creadas y ayudándole en la tarea de la interpretación.
