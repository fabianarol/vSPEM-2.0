La herramienta da soporte tecnológico al paradigma VRP para la definición de procesos ricos en variantes en el contexto de Desarrollo Global de Software, en el que es clave para poder gestionar las similitudes y diferencias de cada organización y sus subproyectos y así facilitar su aplicación en cada una de ellas. Se trata de un plugin de Eclipse Standalone denominado vEPFC (Variant Eclipse Process Framework Composer) que permite el soporte no sólo a los elementos para la definición de procesos software sino también a la variabilidad, de acuerdo a la extensión vSPEM.
