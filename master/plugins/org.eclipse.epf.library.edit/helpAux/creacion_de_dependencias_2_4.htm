2.4 Creación de dependencias

Top  Previous  Next

En este apartado se aprenderá a generar dependencias con los puntos de variación y las variantes. Para la generación de dependencias se realizará desde la edición de los elementos vSPEM generados, aunque se ha visto que también es posible realizarlo desde la creación de elementos vSPEM.

Para editar un elemento se debe pulsar sobre este con el botón derecho y seleccionar la opción Edit tal y como se vio anteriormente. Dentro del menú de edición se selecciona el tipo de dependencia que se quiere añadir, en este caso será variant2variant exclusiva.

Pulsando sobre el botón de añadir (Add) se abrirá una ventana para añadir Variantes.

Se añade VarPhase A a esta dependencia. Una vez pulsado sobre el botón Ok la dependencia se guardará de forma satisfactoria, es conveniente refrescar (Refresh) el proyecto al realizar cambios de estos tipos para que los demás elementos sean conscientes del cambio. De la misma forma podemos usar los botones para borrar las dependencias (Delete).
